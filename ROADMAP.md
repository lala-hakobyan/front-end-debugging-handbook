# Project Roadmap

The following features are likely to be included in future updates of this handbook. Includes features suggested by the community:

## First Edition
The current edition aims to introduce new deep-dive sections like:

- Debugging Memory Leaks in the **Memory** panel
- Debugging real-time performance with **Performance Monitor**
- Playwright MCP server workflows

## Second Edition
Considering readers already have deep knowledge in fundamental and modern debugging tools, this version is expected to pick specific complex issues on front-end applications and debug them step by step using the knowledge and debugging tips gained in the First Edition of the handbook.
