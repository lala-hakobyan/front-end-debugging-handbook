# Project Roadmap

## First Edition Updates
The following features are likely to be included in future updates of this handbook. This includes also features suggested by the community:

- Debugging Memory Leaks in the **Memory** panel.
- Debugging real-time performance with **Performance Monitor**.
- Playwright MCP server workflows.

## Second Edition
Considering readers already have deep knowledge of fundamental and modern debugging tools, this version is expected to pick specific complex issues on front-end applications and debug them step by step using the knowledge and debugging tips gained in the First Edition of the handbook.
